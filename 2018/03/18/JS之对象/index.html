对象是什么

JavaScript是一种面向对象(object-oriented)的语言。对象是什么呢？对象就是某种东西。对于JavaScript，它处理的对象都在Web浏览器中，比如窗口、表单，以及如按钮和复选框的表单元素。对象是一种复合值：它将很多值(原始值或者其他对象)聚合在一起，可通过名字访问这些值。对象也可看做是属性的无序集合，每个属性都是一个名/值对。JavaScript对象是动态的，可以新增属性也可以删除属性。对象也是可变的，我们通过引用而非值来操作对象。

如果不同的变量名指向同一个对象，那么它们都是这个对象的引用，也就是说指向同一个内存地址。修改其中一个变量，会影响到其他的变量。如果取消某一个变量对于原对象的引用，不会影响到另一个变量。这种引用只局限于对象，如果两个变量指向同一个原始类型的值，这时的变量是值的拷贝。

创建对象

可以通过对象直接量、关键字new和Object.create()函数来创建对象。
对象字面量表达式以{开头，以}结尾，语法: { 属性名: 属性值, 属性名: 属性值, …… }。
var x = {a: 'Hi'}
这个对象内部包含一个键值对(又称为”成员”)，a是“键名”(成员的名称)，字符串Hi是“键值”(成员的值)。键名与键值之间用冒号分隔。如果对象内部包含多个键值对，每个键值对之间用逗号分隔。

对象的所有键名都是字符串，所以加不加引号都可以。如果键名不符合标识名的条件就必须加上引号，否则会报错。属性可以动态创建，不必在对象声明时就指定。

属性

属性包括名字和值。属性名可以是包含空字符串在内的任意字符串，但对象中不能存在两个同名的属性。属性值可以被指定为任意类型的值，包括数组或其他的对象，属性值甚至还可以是一个函数。如果一个属性的值为函数，通常把这个属性称为“方法”，它可以像函数那样调用。可以随时新增对象的属性，而不用在定义对象的时候，就定义好属性。可以把任意类型的值、对象或者函数赋值给对象的属性。

通过点(.)或方括号([])运算符可以获取属性的值。在点运算符之后书写属性名，就能够读取相应的属性值。

如果属性的值是一个对象，可以通过多次点运算来读取其属性。

在赋值表达式的左侧书写属性访问表达式的话，就可以将相应的值赋给该属性。

如果赋值给尚不存在的属性名，则新建属性并对其赋值。

如果使用方括号运算符，键名必须放在引号里面，否则会被当作变量处理。object['key']与object[key]是不同的，key加了引号是字符串，不加引号则是变量。数值键名不能使用点运算符(会被当成小数点)，只能使用方括号运算符。点运算符和方括号运算符，既可以用来读取值，也可以用来赋值。

想要查看一个对象本身的所有属性，可以使用Object.keys方法。

删除属性的唯一方法是使用delete，将属性值设置为undefined或null只是移除了与属性相关的值，并没有真正删除属性本身。delete命令用于删除对象的属性，删除成功后返回true，删除一个不存在的属性，也会返回true，只有在该属性存在且不能删除的情况下delete命令才会返回false。delete命令只能删除对象本身的属性，无法删除继承的属性。

in运算符用于检查对象是否包含某个属性(检查的是键名，不是键值)如果包含就返回true，否则返回false。但是它分辨不出哪些属性是对象自身的，哪些属性是继承的。就算是继承的属性，它也会返回true。
