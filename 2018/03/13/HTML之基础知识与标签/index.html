HTML之基础知识与标签

发表于 2018-03-13 | 更新于: 2018-03-13 | 分类于 HTML | | 本文总阅读量 次
字数统计: 939 | 阅读时长 ≈ 3

HTML即 Hypertext Markup Language 是 超文本标记语言 的缩写。HTML是网页内容的载体。内容就是网页制作者放在页面上想要让用户浏览的信息，可以包含文字、图片、视频等。

HTML标签由英文尖括号 < 和 > 括起来的，如<html>就是一个标签。html中的标签一般都是成对出现的，分为开始标签和结束标签。结束标签比开始标签多了一个/。标签与标签之间是可以嵌套的，但先后顺序必须保持一致，如：<div>里嵌套<p>，</p>必须放在</div>的前面。

HTML标签不区分大小写，<h1>和<H1>是一样的，但建议小写，因为大部分程序员都以小写为准。

HTML文件有自己固定的结构。<html></html>称为根标签，所有的网页标签都在<html></html>中。<head>标签用于定义文档的头部，它是所有头部元素的容器。
编写HTML时要把头部和页面主体body分开。在<body>和</body>标签之间的内容是网页的主要内容，在这里的标签中的内容会在浏览器中显示出来。

<head>标签的作用: 文档的头部描述了文档的各种属性和信息，包括文档的标题等。绝大多数文档头部包含的数据都不会真正作为内容显示给读者。

<title>标签：在<title>和</title>标签之间的文字内容是网页的标题信息，它会出现在浏览器的标题栏中。网页的title标签用于告诉用户和搜索引擎这个网页的主要内容是什么，搜索引擎可以通过网页标题，迅速的判断出网页的主题。每个网页的内容都是不同的，所以每个网页都应该有一个独一无二的title。

什么是代码注释？<!-- 注释 -->代码注释的作用是帮助程序员标注代码的用途, 但是注释代码是不会在网页中显示出来的。

HTML里面有很多标签，比如<p>、<br>、<q>、<em>、<a>、<img>。这里着重介绍一下<a>与<img>标签。

a标签
使用id属性为<a>标签创建目标，id属性可以让你直链到这些元素。id所指定的标识符在页面中必须是唯一的。
使用target打开新窗口
使用target属性，你可以定义被链接的文档在何处显示。
如果使用_blank作为目标，浏览器就会打开一个新窗口显示页面。
如果使用_self作为目标，浏览器会在当前页面打开。

img 标签
如何在网页里添加图片呢？这时就需要<img>标签来帮忙啦。<img>标签可以在当前网页中添加一张图片。如此例所示：<img src="images/drink.gif">, src属性指定了在web页面上显示的图像文件的位置。src属性不只可以用于相对链接，还可以在src属性中放入URL。如果要指向另一个不同网站上的图像，通常会使用这个图像的URL。如果图片因为各种原因无法显示，我们可以利用img的alt属性为访问者提供一些指示，告诉他们图像里包含些什么信息。使用alt属性是为了给那些不能看到你文档中图像的浏览者提供文字说明。而使用title属性提供非本质的额外信息。通过使用width和height，我们还可以为<img>标签调整大小。宽度和高度都使用像素数来指定。
