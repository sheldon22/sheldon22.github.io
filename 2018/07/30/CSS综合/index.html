  2. 使用媒体查询，针对不同的屏幕大小来设置。比如

  3. 可以使用百分比布局，但是高度无法跟宽度配合，Pass…

  4. 可以使用动态Rem来进行等比缩放。Rem是什么？Rem是以根元素的字体大小为基准的CSS单位。众所周知，在页面中，html是根元素。所以可以得出rem = html font-size。得出这个结论之后，我们就可以通过更改html的字体大小来间接的更改rem了。我们可以将html的font-size设置成跟页面宽度一致。由于rem=html的font-size。所以rem也等于页面宽度了。然后通过设置rem来缩放页面。达到等比缩放的效果，适配各种手机端的页面。
